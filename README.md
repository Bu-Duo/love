# 给陈多多公主的小宇宙

这是一个充满爱意的互动网页，为陈多多公主打造的专属浪漫体验。页面通过精美的动画和交互效果，展示了各种温馨的文字，形成一个动态的心形图案。

## 项目简介

这个项目是一个纯前端实现的互动页面，通过HTML、CSS和JavaScript创建了一个视觉效果优美、交互体验流畅的爱情表达工具。页面设计注重细节，包含了精心调校的动画效果和响应式布局，确保在各种设备上都能完美展示。

## 使用方法

1. 确保 `warm-001.mp3` 文件与 `love.html` 在同一目录下
2. 直接在浏览器中打开 `love.html` 文件
3. 点击页面中央的按钮开始体验
4. 可以点击和拖拽爱心文字气泡进行互动
5. 点击右下角的「再来一次」按钮可以重新生成爱心

## 功能说明

- **爱心文字动画**：页面加载后，点击按钮会显示由温馨文字组成的动态爱心
- **交互式气泡**：每个文字气泡都可以点击和拖拽，拖拽后会弹回原位
- **悬浮效果**：文字气泡会有轻微的上下浮动效果，模拟呼吸感
- **背景音乐**：页面包含背景音乐，增强浪漫氛围
- **响应式设计**：适配各种屏幕尺寸，在手机和电脑上都能良好显示
- **重新生成**：可以随时点击右下角按钮重新生成爱心图案

## 技术实现

### 心形图案生成
- 使用经典的心形曲线公式：`x = 16 * sin(t)^3`, `y = 13 * cos(t) - 5 * cos(2t) - 2 * cos(3t) - cos(4t)`
- 通过随机因子增加文字气泡的自然分布
- 使用CSS变量存储每个气泡的位置信息

### 动画效果
- CSS动画实现文字气泡的上下浮动效果
- 错峰显示每个气泡，营造「绽放」的视觉效果
- 平滑的过渡动画确保交互体验流畅

### 交互设计
- 使用Pointer Events API实现跨设备的拖拽功能
- 拖拽时暂停动画，释放后恢复，提供更好的交互反馈
- 点击气泡时自动提升层级，确保视觉优先级

### 视觉设计
- 渐变色背景营造柔和氛围
- 毛玻璃效果增强现代感
- 精心调校的颜色方案和阴影效果提升质感
